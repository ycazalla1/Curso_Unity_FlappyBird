## Objetivo del curso
- Aprender a controlar el motor gráfico Unity Engine a un nivel básico
- Crear tus primeros scripts en C#
- Descubrir las posibilidades a la hora de crear los gráficos de un juego
- Crear una copia exacta del videojuego Flappy Bird

## Estructura del curso
Este curso está divido en las siguientes partes:

### Parte I
1. Introducción (Instalación y creación del proyecto)
2. Conceptos Básicos de Unity
3. Desarrollo del juego I

### Parte II
4. Desarrollo del juego II
