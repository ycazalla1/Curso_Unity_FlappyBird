# Descargar gráficos del juego
Como este juego este famoso podemos conseguir los gráficos (sprite sheet) de internet. En mi caso los obtengo de [The Spriters Resource](https://www.spriters-resource.com/mobile/flappybird/asset/59894/).

<div>
  <b>Spritesheet</b>
  <p>
    Es una única imagen que agrupa múltiples fotogramas (sprites) organizados en cuadrícula, 
    usada en videojuegos 2D para crear animaciones secuenciales y optimizar el rendimiento. 
    Dentro de estos suelen estar los fondos, personajes, obstáculos, etc.
  </p>
  <img width="512" height="512" alt="image" src="https://github.com/user-attachments/assets/764fdbb4-f1d7-41a8-ae27-f14230fd3764" />
</div>

Se descarga el sprite sheet y, dentro del proyecto Unity, lo insertamos dentro de la carpeta Assets. Una vez añadido, podremos verlo en la ventana de Project > Assets.
